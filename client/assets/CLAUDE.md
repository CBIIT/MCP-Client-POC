# client/assets/

## Purpose
Static assets for the client application including fonts, images, and stylesheets.

## Contents

### fonts/
- **Montserrat font family** (regular and bold weights)
- Multiple formats: `.eot`, `.svg`, `.ttf`, `.woff` for browser compatibility
- Used for application typography

### images/
- **UI icons:** paperclip, plus, bars, upload, chevron, flag
- **Logo:** Main application logo
- **footer/:** Social media icons (Facebook, Instagram, LinkedIn, X/Twitter, YouTube)
- SVG format for scalability

### styles.css
- Global CSS styles
- Custom properties and Bootstrap overrides
- Chat-specific styling
- Component styling

## Contribution to Project
Provides visual identity and user interface elements for the chat application. Fonts ensure consistent typography, icons enable intuitive UI interactions, and styles define the application's look and feel.
